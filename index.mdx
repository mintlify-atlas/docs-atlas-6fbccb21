Ant Design (`antd`) is an enterprise-class UI design language and React component library. It ships over 80 high-quality components — from basic inputs and layout helpers to complex data tables and date pickers — all styled with a cohesive design token system that supports light/dark themes, compact density, and full RTL layouts out of the box.

## Get started in three steps

<Steps>
  <Step title="Install antd">
    Add the package to your React project using your preferred package manager.

    <CodeGroup>
    ```bash npm
    npm install antd
    ```
    ```bash yarn
    yarn add antd
    ```
    ```bash pnpm
    pnpm add antd
    ```
    </CodeGroup>
  </Step>
  <Step title="Import and use a component">
    Import any component directly from `antd`. Styles are applied automatically via CSS-in-JS — no separate CSS import needed.

    ```tsx
    import { Button, DatePicker } from 'antd';

    export default function App() {
      return (
        <div>
          <Button type="primary">Hello Ant Design</Button>
          <DatePicker />
        </div>
      );
    }
    ```
  </Step>
  <Step title="Customize the theme (optional)">
    Wrap your app in `ConfigProvider` to override design tokens globally or per-component.

    ```tsx
    import { ConfigProvider, Button } from 'antd';

    export default function App() {
      return (
        <ConfigProvider theme={{ token: { colorPrimary: '#00b96b' } }}>
          <Button type="primary">Custom theme</Button>
        </ConfigProvider>
      );
    }
    ```
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="Design Token system" icon="droplet" href="/advanced/customize-theme">
    Seed tokens derive a full palette automatically. Override globally or per component with `ConfigProvider`.
  </Card>
  <Card title="Dark and compact modes" icon="moon" href="/advanced/customize-theme">
    Switch between `defaultAlgorithm`, `darkAlgorithm`, and `compactAlgorithm` at runtime.
  </Card>
  <Card title="Internationalization" icon="globe" href="/advanced/internationalization">
    Built-in locale support for 150+ languages. Pass any locale to `ConfigProvider`.
  </Card>
  <Card title="Server-side rendering" icon="server" href="/advanced/server-side-rendering">
    Full SSR support with inline style extraction or static CSS export via `@ant-design/cssinjs`.
  </Card>
  <Card title="TypeScript-first" icon="code" href="/components/button">
    Every component ships with complete TypeScript definitions. Use `GetProps` to extract prop types.
  </Card>
  <Card title="Framework integrations" icon="layer-group" href="/frameworks/use-with-next">
    Dedicated guides for Next.js, Vite, Umi, Rsbuild, and more.
  </Card>
</CardGroup>
